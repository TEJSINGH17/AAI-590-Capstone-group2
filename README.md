# AAI-590 Capstone Group 2

OmniView AI is a real-time traffic sign detection and blind spot monitoring system with an augmented reality heads-up display, built on YOLOv8 and NVIDIA DeepStream.

| Folder | Description |
|---|---|

## Training Workflow

1. Place the raw LISA dataset under `training_data/Lisa/`
2. Convert annotations to YOLO format:

```bash
python -m training.prepare_lisa \
  --lisa_root training_data/Lisa \
  --out_root training/lisa_yolo
```

3. Train YOLOv8 on the converted dataset:

```bash
python -m training.train_yolov8 \
  --data training/lisa_yolo/dataset.yaml \
  --base_model models/yolov8n.pt
```

4. Run inference on a test video:

```bash
python test_mp4_yolov8.py \
  --model trained_models/yolov8n_lisa_best.pt \
  --input test_data/output_ds_1.mp4 \
  --output runs/annotated/out.mp4
```
